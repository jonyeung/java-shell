User Stories from Product Backlog that we will address in this sprint:
a) As a user, I can list all files in a directory and its subdirectories. (ls)
b) As a user, I can move files from one directory to another. (mv)
c) As a user, I can copy files from one directory to another. (cp)
d) As a user, I can get the contents from a URL (curl)
e) As a user, I can recall specific commands based on their history 
	number. (history !number)
f) As a user, I can see which lines in a file match a regex. (grep)
g) As a user, I can view the contents of files (cat)
